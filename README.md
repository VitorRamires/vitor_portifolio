<h3>🎯 Objetivo</h3>

Este projeto foi criado com o objetivo de apresentar minhas experiências profissionais, habilidades técnicas e projetos desenvolvidos como desenvolvedor.

É uma vitrine funcional do que sei fazer, com foco em tecnologias modernas e boas práticas de desenvolvimento.

<br />

<h3>🛠️ Construído com:</h3>

<ul>

  <li>Typescript - linguagem base do projeto</li>

  <li>React.js - Framework</li>

  <li>Motion Frame - Biblioteca de animação</li>

</ul>

<br />

<h3>🚀 Como rodar:</h3>

<pre><code>

npm install

# Inicie o servidor de desenvolvimento
npm run dev
</code></pre>
